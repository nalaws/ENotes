# ENotes
电子便签

## 主要功能

* [x] 创建记事本
* [x] 记事本加密
* [x] 记事本添加标签
* [x] 记事本搜索
* [x] 设置头像
* [x] 设置座右铭
* [x] 二维码识别和生成
* [x] 文本识别
* [x] 日历(支持农历)
* [x] 设置日程
* [x] 设置纪念日
